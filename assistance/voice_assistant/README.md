# AI Voice Assistant for Visually Impaired Users

An intelligent voice assistant that understands it's helping visually impaired individuals and provides contextually appropriate responses.

## Features
- Voice-activated AI conversations
- Context-aware responses for visually impaired users
- Speech recognition and text-to-speech integration
- Maintains conversation history and context
- Accessibility-focused interaction patterns

## Components
- `voice_assistant.py` - Main assistant class
- `speech_recognition_handler.py` - Voice input processing
- `conversation_manager.py` - Context and history management
- `accessibility_prompts.py` - Specialized prompts for VI users
- `config.py` - Configuration settings

## Usage
The assistant will:
1. Listen for voice commands
2. Process speech using AI with VI-aware context
3. Provide helpful, accessible responses via TTS
4. Remember conversation context for follow-up questions
