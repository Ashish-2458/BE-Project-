# Simple Voice Assistant

A clean, simple voice assistant that listens to user speech and responds using Gemini AI.

## Features
- Speech-to-text using Google Speech Recognition
- AI responses using Gemini 2.0 Flash
- Text-to-speech for AI responses
- Simple conversation flow

## Usage
```bash
python voice_assistant.py
```

## How it works
1. User speaks
2. Speech converted to text
3. Text sent to Gemini AI
4. AI response converted to speech
5. Repeat
