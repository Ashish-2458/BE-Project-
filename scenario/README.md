# Scenario Feature Development

This folder contains experimental features and scenarios for the OneVision assistive vision system.

## Purpose
- Develop and test new features in isolation
- Prototype different scenarios and use cases
- Test integrations before adding to main system

## Structure
Each feature will have its own subfolder with:
- Implementation files
- Test scripts
- Documentation
- Configuration files

## Development Workflow
1. Create feature in scenario folder
2. Test and refine functionality


python main.py
